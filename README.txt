To launch program, click the launch.bat shortcut.
Alternatively, the program can be run in a terminal with the .\gradlew run command.

To change default binary file, go to app\src\main\resources and replace dk.osm.obj with another .osm file.
Other .osm or .obj files can be added to the app\data folder, or to a folder named "data" in the path of the app.jar file.
